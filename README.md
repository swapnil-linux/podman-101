# Containers with Podman

A hands-on guide to learning containers using Podman — from the fundamentals of containerization to advanced topics like multi-container applications, custom networking, persistent storage, and pod orchestration. Updated for **Podman 5.x** with Netavark networking, Aardvark-DNS, and cross-platform support (Linux, macOS, and Windows).

## What You'll Learn

* **Container fundamentals** — what containers are, how they differ from VMs, and the OCI standards behind them
* **Podman basics** — installing Podman, running your first container, managing container lifecycle
* **Web applications** — deploying and exposing containerized web apps with port forwarding
* **Images and layers** — how container images work, searching registries, pulling and managing images
* **Building images** — writing Containerfiles, multi-stage builds, and pushing to registries like Quay.io
* **Networking** — Netavark bridge networks, DNS-based container name resolution, rootless networking with pasta
* **Persistent storage** — data volumes, bind mounts, and SELinux considerations
* **Pods** — creating and managing pods, generating Kubernetes YAML, Quadlet systemd integration
* **Compose** — using `podman compose` with standard Compose files

## Prerequisites

* A Linux system (Fedora, RHEL 9, Ubuntu 22.04+), or macOS/Windows with Podman installed
* Basic familiarity with the Linux command line
* Root or sudo access (for rootful exercises); most exercises also work rootless

## How to Use This Guide

Each chapter combines theory with hands-on practice labs. Work through the chapters in order — each builds on concepts from the previous one. Practice files are prefixed with `practice-` and contain step-by-step exercises you should follow along with on your own system.
